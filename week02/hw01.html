Translate Java to C++ - Your First C++ Project!

üéØ Assignment Goal: This assignment helps you learn C++ syntax by translating a Java program. You'll work with both classes and a main program!

üìñ Assignment Overview

For this assignment, you will translate a provided Java program to C++. The purpose of this assignment is to help you learn the syntax for C++ statements and understand how Java classes convert to C++'s header/implementation file structure.

üéÆ What You're Building

You'll be converting a Java program that consists of:

  • Monster Class - A class that represents a game monster with attributes and methods
  • KillMonster Program - A main program that creates and uses Monster objects
üí° Remember: In C++, classes are split into two files: a .h header file (declarations) and a .cpp implementation file (definitions). The main program goes in its own .cpp file.

üìù Requirements

Your C++ solution must include the following files:

Required Files (3 total):

  • Monster.h - Header file containing:
    • Class declaration
    • Member variable declarations
    • Method prototypes (signatures)
    • Include guards
  • Monster.cpp - Implementation file containing:
    • Method implementations (the actual code)
    • #include "Monster.h" at the top
  • KillMonster.cpp - Main program file containing:
    • int main() function
    • Program logic that uses the Monster class
    • #include "Monster.h" at the top

üîç Key Translation Points

  • Convert boolean to bool
  • Convert String to string (and include <string>)
  • Change System.out.println() to cout << ... << endl;
  • Use #include <iostream> for I/O operations
  • Remember to add using namespace std; or use std:: prefix
