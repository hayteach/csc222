ğŸ‘‹ Hello World Showdown: Java vs C++

Let's start with the classics! Notice how C++ is more... minimalist? ğŸ˜

The Java Way (What You Know)

class Compare
{
  public static void main(String[] args)
  {
    // All wrapped in a class
    System.out.println("Hello, World!");
  }
}

The C++ Way (Simpler!)

#include <iostream>  // Bring in I/O library
using namespace std;   // Use standard namespace

int main()
{
  // No class needed! 
  cout << "Hello, World!" << endl;
  return 0;  // 0 means success
}

ğŸ” Key Differences Explained

  • #include <iostream> - Like Java's import, but literally copies code into your file at compile time
  • using namespace std; - Lets you use cout instead of std::cout (saves typing!)
  • int main() - No need for "public static void" or class wrapper! Just a simple function
  • cout << - Pronounced "see-out", uses << operator to send data to console
  • endl - "End line" - moves to next line (like \n)
ğŸ“ Professor's Tip: In Java, everything must be in a class because it's "purely object-oriented." C++ is an "extension" of C, so it supports both procedural (functions) and object-oriented (classes) programming. Best of both worlds!

ğŸ¨ Putting It All Together

Let's see a real program that uses everything we've learned! This program collects names and ages in a loop until the user quits.

Java Version (What You Know)

import java.util.Scanner;

class Basic {
    public static void main(String[] args) {
        Scanner kb = new Scanner(System.in);
        String name;
        int age;
        boolean keepGoing = true;
        
        while (keepGoing) {
            System.out.print("Enter name: ");
            name = kb.nextLine();
            System.out.print("Enter age: ");
            age = kb.nextInt();
            kb.nextLine(); // Flush the line
            
            System.out.println("Hello, " + name + 
                             "! You are " + age + " years old.");
            
            System.out.print("Continue? (yes/no): ");
            String answer = kb.nextLine();
            if (!answer.equals("yes")) {
                keepGoing = false;
            }
        }
        System.out.println("Goodbye!");
    }
}

C++ Version (Your New Skill!)

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name, answer;
    int age;
    bool keepGoing = true;
    
    while (keepGoing) {
        cout << "Enter name: ";
        getline(cin, name);
        cout << "Enter age: ";
        cin >> age;
        cin.ignore(); // Flush the line
        
        cout << "Hello, " << name 
             << "! You are " << age << " years old." << endl;
        
        cout << "Continue? (yes/no): ";
        getline(cin, answer);
        if (answer != "yes") {
            keepGoing = false;
        }
    }
    cout << "Goodbye!" << endl;
    return 0;
}
