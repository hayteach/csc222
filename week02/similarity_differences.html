üì¶ Importing Libraries

Both languages allow you to import external code, but they use different syntax.

Purpose Java C++
Standard Libraries import java.util.Scanner; #include <iostream>
Your Own Classes If in same folder, no import needed #include "ClassName.h"

Example: Using Standard I/O

// Java
import java.util.Scanner;

// C++
#include <iostream>
#include <string>

Class creation differs significantly between Java and C++.

Java C++

Single File Approach

    // Attributes and methods
    // all in one file
}

Two File Approach

  • .h file - Header with declarations
  • .cpp file - Implementation with code

üíª Creating Console Programs

    public static void main(String[] args) {
        // program code
    }
}

    // program code
    return 0;
}
