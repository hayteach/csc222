Writing a Console Program

A Console program in C++ is simpler to code than one in Java due to the quirkiness of the Java language. The following table provides a comparison between the program structure of a Console program in Java versus C++

Java C++ Discussion

For file named: Compare.java

class Compare
{
 public static void main(String[] args)
 {
      // program code goes here
 }
}

For c++, filename doesn't matter as long as end with ".cpp" - compare.cpp

int main()
{
 // program code goes here
}

Java, a purely object-oriented programming language, your console program has to be encapsulated in a class that has the same name as the file. C++, in contrast, is an object-oriented extension to the C programming language and as such supports both object-oriented programming and procedural programming.

Because of this, your C++ filename can be anything you want and your console program only has to have a function (like a method in Java) that is named "main" and returns an integer.

Console Output

Unlike Java, there is no pre-built console output function or method in the base language.

Java C++ Discussion

To write output to the console, we use System.out.print...

class Compare
{
  public static void main(String[] args)
  {
    // print to console
    System.out.println("Hello");
  }
}

For C++, we need to include the "iostream" library (not "import" which is how we bring in libraries in Java)

#include<iostream>
int main()
{
  // program code goes here
  cout << "Hello" << endl;
}

In Java, the standard,  unformatted output, is done using the "print" and "println" methods in "System.out"

In C++, the standard console output is done using the "cout" command which is part of the "iostream" library

The format for console output is:

cout << "whatever you want to print to the screen" << "anything else you want to add"

To print multiple items to the screen in Java:

String name = "John";
System.out.println("Hello " + name);

In C++, we can't use the "+" sign to concatenate things together, we have to use the "<<" symbol to do this. 

string name = "John";
cout << "Hello " << name << endl;

In C++, we can use the "endl" constant to move the output to the next line ("endl" stands for end of line). You can also use the "\n" ; "\t"; "\\" and other similar escape sequences just like in Java.

Console Input

For collecting console (keyboard) input ... 

Java C++ Discussion

We need to import the Scanner utility and create an instance. Then we can read the different types of data using the .next(), .nextLine(), .nextInt(), ... methods.

import java.util.Scanner;
class Compare
{
  public static void main(String[] args)
  {
    Scanner kb = new Scanner(System.in);
    System.out.println("Please enter your name: ");
    String name = kb.next();
    System.out.println("Your name is " + name);
  }
}

For C++, the console input utility, "cin", is part of the "iostream" library

#include<iostream>
int main()
{
  // program code goes here
  cout << "Please enter your name: ";
  string name;
  cin >> name;
  cout << "Your name is " << name;
}

In C++, the "cin" utility takes whatever is in the input stream (typed into the keyboard) and puts it into the variable the ">>" symbol points to. NOTE: "cin" expects the appropriate data type being entered.

To collect a number:

int age;
System.out.println("Please enter your age: ");
age = kb.nextInt();
System.out.println("Age + 5 is " + (age + 5));

In C++

int age:
cout << "Please enter your age: ";
cin >> age;
cout << "Age + 5 is: " << age+5 << endl;

In C++, when asked the age, if you try to enter a string like "six", the program will read an age of "0" since there is no integer there.

To collect a line of text with more than 1 word:

name = kb.nextLine();

 

getline(cin, name);

In C++, cin will only get 1 value so a word or value. To read an entire line of code you need to use the "getline()" method which takes 2 parameters, the source for your input (cin) and the variable where what you read in is stored

How to flush the input line:

kb.nextLine();

 

cin.clear(); //this clears any error in cin
cin.ignore(); //this ignores the rest of the line

Often when you are reading console input in Java, we had the problem where the cursor was still on the previous line and this would cause issues especially in loops.

Variable Declarations

Declaration of primitive variables between Java and C++ is mostly identical

Java C++ Discussion

The following primitives are the same:

int age;
double price;
char symbol;

The other primitives that we haven't used as much include:
byte, short, long, float

The equivalent C++ declarations are the same:

int age;
double price;
char symbol;

The other primitives that we won't use as much include:
byte, short, long, float
unsigned versions of char, short, int and long
long double

Typical variables defined by "int, double, ..." are signed meaning they have negative and positive values. In C++ we can declare "unsigned" versions which only have positive values (so we have twice as many positives as compared to a signed version)

To declare a boolean:

boolean isDone = false;

In C++

bool isDone = false;

for booleans in C++, the declaration is "bool" and not "boolean" as it is in Java.

Strings

At a surface level, Strings are nearly identical between Java and C++

Java C++ Discussion

String declaration is done as follows

String name = "John";

The equivalent C++ declaration is:

string name = "John";

For C++, there are 2 different string libraries. If needed, you should only use/include the "string" library by including:

#include <string>

For more information on strings in C++: string - C++ Reference (cplusplus.com) 

Branching

Branching is used for decision making and there are 2 basic structures that we use for this:

  • The IF, IF-ELSE, IF-ELSEIF-ELSE structures
  • The SWITCH-CASE structure

The code for these is identical between Java and C++

Looping

The 3 basic looping structures we covered in Java:

  • for loops
  • while loops
  • do-while loops

Are the same in C++
