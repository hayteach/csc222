## iFrame Usage

To embed these pages in an iframe, use the following format:

```html
<iframe src="https://yourusername.github.io/csc222/week05/index.html" width="100%" height="600" frameborder="0"></iframe>
```

Replace `yourusername` with your actual GitHub username and adjust the page name as needed.
