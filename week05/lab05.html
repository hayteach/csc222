Overview

Add a method remove(n) to the Queue ADT that discards the first n entries from the queue. Ensure it handles edge cases like n > size and n <= 0.

Instructions

  • Update the Queue interface to declare void remove(int n) (or equivalent in C++ signatures).
  • Implement the method in the array-based queue implementation and any linked-list implementation used in class.
  • Update the tester program to exercise remove(n) for typical and edge cases.

Submission
