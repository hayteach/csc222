Write a program that:

  1. Prompts the user to input the value of an integer.
  2. Store that input into an "int" called "myVar"
  3. Create a pointer, "iptr" and store the address of "myVar" in "iptr"
  4. Print out the value of the integer "myVar", the address of "myVar" (stored in iptr) and the value of the integer stored at the address that "iptr" is pointing to.
  5. Create an integer array, "myArray" of size "myVar" and fill it with integers from 1 to "myVar"
  6. Print out the values in "myArray"
  7. Create a pointer to "myArray" called "ptrToMyArray"
  8. Print out the values in "myArray" using "ptrToMyArray" to access each value.
  9. Make sure that your program properly deletes everything and frees memory upon exit.
