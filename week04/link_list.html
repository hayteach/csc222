Starter Code

// Node.h
#ifndef NODE_H
#define NODE_H

template<class ItemType>
class Node {
public:
    ItemType data;
    Node<ItemType>* next;
    Node(const ItemType &d, Node<ItemType>* n = nullptr) : data(d), next(n) {}
};

#endif

// LinkedBag (interface sketch)
template<class ItemType>
class LinkedBag {
public:
    bool add(const ItemType &item);
    bool remove(const ItemType &item);
    bool contains(const ItemType &item) const;
    int getCurrentSize() const;
    bool isEmpty() const;
};

Practice & Tips

  • Begin with implementing basic operations: insert at head, insert at tail, traverse and print.
  • Test edge cases: empty list, single-item list, removing head/tail, removing non-existing items.
  • Use Valgrind while testing to catch memory leaks and invalid accesses early.
  • Work through a few easy LeetCode or GeeksforGeeks problems to build confidence.
